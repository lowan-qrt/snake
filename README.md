## Installation
1. Clone the repository to your local machine:
```bash
git clone https://github.com/lowan-qrt/snake.git
```

2. Navigate into the project directory:
```bash
cd snake
```

3. Install required Python modules using the requirements file:
```bash
pip install -r requirements.txt
```

4. Run de the game:
```bash
python jeu.py
```
